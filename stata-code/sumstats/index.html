    Summary statistics tables

    Making tables of summary statistics with Stata is now easy! Here's an example using sumstats:

    1
2
3
4
5
    sysuse auto.dta , clear
sumstats  ///
  (price mpg if foreign == 0) ///
  (price displacement length if foreign == 1) ///
  using "test.xlsx" , replace stats(mean sd)

    sumstats is available on SSC and is open for development on GitHub. Submit bugs and feature requests here. If you like sumstats, be sure to visit my homepage and Stata boilerplate code.
